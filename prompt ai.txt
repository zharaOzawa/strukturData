Buatlah program Python untuk mengelola data mahasiswa (NIM dan Nama) menggunakan array dengan kapasitas statis 10. Jangan gunakan fungsi built-in list seperti .append, .pop, atau .insert. Gunakan perulangan (looping) untuk menggeser elemen saat menambah data (Insert) dan menghapus data (Delete) pada posisi tertentu. Menu harus mencakup: tambah/hapus di awal, di akhir, dan di posisi tertentu, serta hapus berdasarkan NIM, tambahkan juga variabel count agar terlihat bagus
